디자인 원칙

① Neutral Gray Palette

Blue 강조 컬러는 사용하지 않습니다. Brand Primary 는 --c-neutral-700 #374151. 상태 표시(success/warning/danger)에만 채도 있는 컬러를 사용합니다.

② Token First

모든 색상·간격·타이포는 tokens.css·semantic.css 토큰 기준. hex 직접 사용 금지 (legacy 호환 영역 제외).

③ Card 구조 통일

모든 카드는 .card + .card__head + .card__body + .card__footer 구조. padding/gap 은 --card-body-padding·--card-body-gap·--card-footer-padding-y 토큰만 사용.

④ Selected 상태는 연한 Gray

버튼·칩의 선택 상태는 dark navy/black 아닌 #F3F4F6 / border #9CA3AF / text #111827 조합.

⑤ Fixed Canvas 1920×1080

WebView 고정 캔버스. 헤더 64px / LNB 96px / 푸터 40px / 3열 (28% · 32% · 40%) 구조.
