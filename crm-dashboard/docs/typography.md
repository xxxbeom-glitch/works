# Typography

폰트: **Pretendard Variable**
웨이트: Medium (500), SemiBold (600), Bold (700)

## Font Weight

| Token | Value | 용도 |
|---|---|---|
| `--font-weight-medium` | `500` | 본문, 라벨 |
| `--font-weight-semibold` | `600` | 서브 헤딩, 버튼 |
| `--font-weight-bold` | `700` | 헤딩, 강조 |

## Font Size

| Token | Value | 용도 |
|---|---|---|
| `--text-xs` | `11px` | 캡션, 뱃지 |
| `--text-sm` | `12px` | 보조 텍스트, 메타 |
| `--text-base` | `14px` | 기본 본문 |
| `--text-md` | `15px` | 약간 큰 본문 |
| `--text-lg` | `16px` | 섹션 제목, 강조 본문 |
| `--text-xl` | `18px` | 카드 헤딩 |
| `--text-2xl` | `20px` | 페이지 서브 타이틀 |
| `--text-3xl` | `24px` | 페이지 타이틀 |
| `--text-4xl` | `28px` | 대형 수치, KPI |
| `--text-5xl` | `32px` | 히어로 수치 |

## Line Height

| Token | Value | 용도 |
|---|---|---|
| `--leading-tight` | `1.2` | 헤딩 |
| `--leading-snug` | `1.4` | 서브 헤딩 |
| `--leading-normal` | `1.5` | 기본 본문 |
| `--leading-relaxed` | `1.6` | 긴 본문 |

## Letter Spacing

| Token | Value | 용도 |
|---|---|---|
| `--tracking-tight` | `-0.02em` | 대형 헤딩 |
| `--tracking-normal` | `0` | 기본 본문 |
| `--tracking-wide` | `0.04em` | 캡션, 라벨 |
| `--tracking-wider` | `0.08em` | 대문자 라벨 |

## 사용 예시

| 역할 | size | weight | leading |
|---|---|---|---|
| 페이지 타이틀 | 24px | Bold | 1.2 |
| 카드 헤딩 | 18px | SemiBold | 1.4 |
| 본문 | 14px | Medium | 1.5 |
| 보조 텍스트 | 12px | Medium | 1.5 |
| 캡션/뱃지 | 11px | Medium | 1.4 |
| 버튼 | 14px | SemiBold | 1 |
| KPI 수치 | 32px | Bold | 1.2 |
